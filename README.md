## 功能列表

- 目前已实现

- [x] 启动游戏
- [x] 领取活动费
- [x] 竞赛挑战
  - [x] 指定挑战
  - [ ] 自动选择
- [x] 社团互动
  - [x] `自动/指定` 请求
- [x] 安排工作
  - [x] 领取奖励
  - [x] `自动/指定` 选择偶像
  - [x] 指定时长
- [x] 商店购买
  - [x] 扭蛋币购买
  - [x] 金币购买[^1]
  - [X] AP购买[^1]
- [x] 领取邮箱礼物
- [x] 领取任务奖励

[^1]: 因MaaFramework框架限制，购买指定物品需要手动修改config.jsonc文件
